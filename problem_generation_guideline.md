# 수능 경향 기반 문제 생성 가이드라인

## 📊 경향성 분석 결과 요약 (2022-2026, 299개 문제)

### 핵심 통계
- **평균 정답률**: 58-61% (연도별)
- **킬러 문항 비율**: 17-20% (매년 일정)
- **과목별 분포**: 기하 33.1%, 미적분 33.1%, 확률과통계 33.8%

### 문항번호별 경향
- **1-15번 (기본)**: 평균 정답률 77.0% - 기본 개념 확인
- **16-22번 (중급)**: 평균 정답률 55.3% - 계산 능력 측정
- **23-28번 (고급)**: 평균 정답률 38.5% - 개념 이해 평가
- **29-30번 (킬러)**: 평균 정답률 15.2% - 창의력 평가

### 문제 형식 경향
- **선택형(계산)**: 68.9% (가장 많음)
- **단답형**: 31.1%

### 함정 유형 경향
- **없음**: 79.9% (대부분 직선적 풀이)
- **복합함정형**: 9.4%
- **부호실수형**: 6.4%
- **특수경우형**: 4.3%

### 출제 의도 경향
- **계산능력측정**: 44.5%
- **기본개념확인**: 26.1%
- **변별력확보**: 15.1%
- **창의력평가**: 14.4%

## 🎯 문제 생성 가이드라인

### 1. 문항번호별 생성 기준

#### 1-15번 (기본 문항)
- **목표 정답률**: 75-85%
- **난이도**: 1등급컷용 ~ 2-3등급용
- **풀이 시간**: 1-3분
- **문제 형식**: 선택형(계산) 80%, 단답형 20%
- **함정**: 없음 (직선적 풀이)
- **출제 의도**: 기본개념확인
- **핵심개념**: 단일개념
- **계산복잡도**: 암산가능 ~ 간단계산
- **풀이단계수**: 2-3단계

**생성 예시**:
```
- 교과서 정의를 직접 적용하는 문제
- 기본 공식 사용 문제
- 단순 계산 문제
```

#### 16-22번 (중급 문항)
- **목표 정답률**: 50-65%
- **난이도**: 4-6등급용
- **풀이 시간**: 3-5분
- **문제 형식**: 선택형(계산) 70%, 선택형(이해) 30%
- **함정**: 조건누락형 또는 부호실수형 가능
- **출제 의도**: 계산능력측정
- **핵심개념**: 단일개념 ~ 융합개념
- **계산복잡도**: 간단계산 ~ 복잡계산
- **풀이단계수**: 3-5단계

**생성 예시**:
```
- 여러 단계 계산이 필요한 문제
- 조건을 해석해야 하는 문제
- 그래프나 표를 활용하는 문제
```

#### 23-28번 (고급 문항)
- **목표 정답률**: 30-45%
- **난이도**: 변별용
- **풀이 시간**: 5-7분
- **문제 형식**: 선택형(계산) 60%, 선택형(추론) 40%
- **함정**: 복합함정형 또는 특수경우형
- **출제 의도**: 변별력확보
- **핵심개념**: 융합개념 ~ 복합개념
- **계산복잡도**: 복잡계산 ~ 매우복잡
- **풀이단계수**: 5-7단계

**생성 예시**:
```
- 여러 개념을 융합한 문제
- 추론이 필요한 문제
- 조건이 복잡한 문제
```

#### 29-30번 (킬러 문항)
- **목표 정답률**: 10-25%
- **난이도**: 킬러용 ~ 슈퍼킬러
- **풀이 시간**: 8-10분
- **문제 형식**: 선택형(추론) 50%, 선택형(증명) 50%
- **함정**: 복합함정형 (여러 함정 동시)
- **출제 의도**: 창의력평가
- **핵심개념**: 복합개념
- **계산복잡도**: 매우복잡
- **풀이단계수**: 7단계 이상

**생성 예시**:
```
- 새로운 접근이 필요한 문제
- 창의적 사고가 필요한 문제
- 여러 단원을 융합한 문제
```

### 2. 과목별 생성 기준

#### 기하 (33.1%)
- **주요 중단원**: 이차곡선, 평면벡터, 공간도형
- **평균 정답률**: 60.0%
- **특징**: 도형과 벡터 개념 중심

#### 미적분 (33.1%)
- **주요 중단원**: 수열의극한, 미분법, 적분법
- **평균 정답률**: 58.8%
- **특징**: 계산과 추론이 복합적으로 요구

#### 확률과통계 (33.8%)
- **주요 중단원**: 확률 (25개, 가장 많음)
- **평균 정답률**: 59.4%
- **특징**: 확률 문제가 가장 빈번하게 출제

### 3. 문제 생성 체크리스트

문제 생성 시 다음 항목을 확인하세요:

#### 필수 확인 사항
- [ ] 문항번호에 맞는 목표 정답률 설정
- [ ] 문항번호에 맞는 풀이 시간 설정
- [ ] 문항번호에 맞는 난이도 등급 설정
- [ ] 문제 형식이 경향에 맞는지 확인
- [ ] 함정 유형이 적절한지 확인
- [ ] 출제 의도가 명확한지 확인

#### 권장 사항
- [ ] 최근 3년(2024-2026) 경향 반영
- [ ] 과목별 주요 중단원 활용
- [ ] 계산능력측정 또는 기본개념확인 중심
- [ ] 복합함정은 23번 이상에서만 사용
- [ ] 1-15번은 직선적 풀이 (함정 없음)

### 5. 품질 검증 기준

생성된 문제는 다음 기준으로 검증하세요:

1. **정답률 일치도**: 목표 정답률 ±5% 이내
2. **풀이 시간**: 목표 시간 ±1분 이내
3. **난이도 등급**: 문항번호에 맞는 등급
4. **함정 적절성**: 문항번호에 맞는 함정 유형
5. **경향성 반영**: 최근 3년 경향과 일치
