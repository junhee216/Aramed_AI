# Linux Cursor AI에게 전달할 Git 설정 메시지

## 복사해서 Linux Cursor에 붙여넣으세요!

```
Git을 사용하여 프로젝트를 동기화하기로 결정했습니다.

## Windows에서 진행할 작업

1. **Git 저장소 초기화** (Windows에서 진행 예정)
   ```bash
   cd C:\Users\a\Documents\Aramed_AI
   git init
   git add .
   git commit -m "Initial commit: Aramed_AI project"
   ```

2. **원격 저장소 설정** (GitHub/GitLab 등)
   - GitHub에 새 저장소 생성 예정
   - 또는 GitLab, Bitbucket 등 사용 가능

3. **원격 저장소에 푸시** (Windows에서)
   ```bash
   git remote add origin [저장소 URL]
   git push -u origin main
   ```

## Linux에서 진행할 작업

**요청 사항:**

1. **필수 도구 설치 확인 및 설치**
   - Git 설치 확인 및 설치 (필요시)
   - Python 3 설치 확인 및 설치 (필요시)
   - Node.js 설치 확인 및 설치 (필요시)
   - pip, npm 설치 확인 및 설치 (필요시)

2. **Git 저장소 클론 준비**
   - ~/Documents/Aramed_AI 디렉토리 생성 (없다면)
   - 또는 원하는 위치 지정

3. **클론 명령어 안내**
   - Windows에서 원격 저장소 설정 완료 후
   - Linux에서 클론할 수 있는 명령어 제공

4. **환경 설정 안내**
   - .env 파일 생성 방법
   - MATHPDF_PATH 환경 변수 설정 방법
   - 필요한 패키지 설치 방법

## 현재 상태

- Windows에서 Git 저장소 초기화 예정
- 원격 저장소 설정 예정
- Linux에서 클론 대기 중
