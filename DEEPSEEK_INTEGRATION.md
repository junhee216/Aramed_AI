# DeepSeek R1 70B 연동 가이드

## 목표
- Linux 컴퓨터의 DeepSeek R1 70B와 Aramed_AI 프로젝트 연동
- 프로젝트 데이터를 활용하여 학습 진행

## 현재 프로젝트 데이터

### 1. 변환된 문제 데이터
- 형식: JSON, Markdown
- 위치: `~/Documents/MathPDF/organized/현우진/기하_2024학년도_현우진_드릴/`
- 파일 예시:
  - `기하_2024학년도_현우진_드릴_P5_문제_deepseek.json`
  - `기하_2024학년도_현우진_드릴_P5_문제_deepseek.md`

### 2. 변환된 해설 데이터
- 형식: Markdown
- 위치: 동일한 디렉토리
- 파일 예시:
  - `기하_2024학년도_현우진_드릴_P5_해설_deepseek.md`

### 3. Notion 데이터
- Notion 데이터베이스에 저장된 문제/해설 정보
- 필드 26 (원리공유문제), 27 (오답시나리오)

## DeepSeek R1 70B 학습 데이터 형식

### 권장 형식
```json
{
  "messages": [
    {
      "role": "user",
      "content": "문제: [문제 내용]\n\n이 문제를 해결해주세요."
    },
    {
      "role": "assistant",
      "content": "해설: [해설 내용]\n\n원리: [원리]\n\n오답 시나리오: [오답 시나리오]"
```

## 연동 방법 제안

### 방법 1: 데이터 변환 스크립트
- 기존 JSON/Markdown 파일을 DeepSeek 학습 형식으로 변환
- 배치 처리로 모든 문제/해설 변환

### 방법 2: Notion에서 직접 추출
- Notion API를 통해 데이터 추출
- DeepSeek 학습 형식으로 변환

### 방법 3: 통합 파이프라인
- Mathpix → LaTeX 변환
- LaTeX → DeepSeek 형식 변환
- DeepSeek 학습 데이터 생성
- Notion 업데이트

## 다음 단계

1. **데이터 형식 확인**
   - DeepSeek R1 70B가 요구하는 정확한 데이터 형식 확인
   - 현재 프로젝트 데이터와의 호환성 확인

2. **변환 스크립트 작성**
   - 기존 데이터를 DeepSeek 학습 형식으로 변환하는 스크립트
   - 배치 처리 지원

3. **학습 데이터 생성**
   - 모든 문제/해설을 학습 데이터로 변환
   - 검증 및 품질 확인

4. **연동 테스트**
   - DeepSeek R1 70B와 연동 테스트
   - 학습 진행 확인
