```
Windows에서 Git 저장소 설정이 완료되었습니다.

## 저장소 정보
- 저장소 URL: [여기에 GitHub 저장소 URL 붙여넣기]
- 예: https://github.com/YOUR_USERNAME/Aramed_AI.git

## 요청 사항

1. **Git 설치 확인 및 설치**
   - Git이 설치되어 있는지 확인
   - 설치되어 있지 않으면 설치 진행
   - 설치 명령어: `sudo apt install -y git` (Ubuntu/Debian 기준)

2. **프로젝트 클론**
   - ~/Documents/Aramed_AI 디렉토리에 클론
   - 또는 원하는 위치에 클론

3. **클론 후 환경 설정**
   - .env 파일 생성 방법 안내
   - MATHPDF_PATH 환경 변수 설정 방법
   - 필요한 패키지 설치 (npm install, pip install -r requirements.txt)

4. **프로젝트 상태 확인**
   - 파일 구조 확인
   - 완료된 작업 확인

클론을 진행해주세요!
```
