### 2. 재사용 가능한 유틸리티 함수 정리 ✅

**생성된 파일:**

#### pdf_utils.py
- `extract_text_from_pdf(pdf_path, page_num=None)`: PDF에서 텍스트 추출
  - 우선순위: pdfplumber > PyPDF2 > pymupdf
  - 전체 또는 특정 페이지 추출 지원
- `find_pdf(base_dir=None, filename_patterns=None)`: 원본 PDF 파일 찾기
  - 여러 검색 경로 자동 탐색
  - 가장 최근 수정된 파일 반환

#### solution_utils.py
- `review_solutions(solutions, check_math_logic=True)`: 해설 데이터 검토
  - LaTeX 수식 괄호 검사
  - 수학적 논리 검사 (삼각함수 범위, 코사인법칙, 사인법칙 등)
- `save_solutions_for_deepseek(solutions, output_dir, base_filename)`: 해설 저장
  - CSV 및 JSON 형식으로 저장
  - 딥시크용 형식으로 변환

**기존 유틸리티:**
- `latex_utils.py`: LaTeX 파싱 유틸리티
- `convert_template.py`: 문제 변환 템플릿
