# Linux Cursor에서 작업 시작하기

## 현재 상황
- Linux Cursor 재설치 후에도 동일한 입력 문제 발생
- **채팅창 입력은 포기하고 복사 붙여넣기로 작업하기로 결정**
- Linux Cursor에서 작업을 이어가고 싶음
- **특히 Mathpix와 Notion 관련 업무**
- **DeepSeek R1 70B와 연동하여 학습시키려고 함**

## 즉시 시작하기

### 1단계: Linux Cursor에 첫 메시지 전달

**`LINUX_START_MESSAGE.md` 파일의 내용을 복사해서 붙여넣으세요!**

또는 텍스트 에디터에 아래 내용을 작성하고 복사 붙이기:

```
안녕하세요! Linux Cursor에서 작업을 이어가려고 합니다.

현재 상황:
- Cursor 재설치 후에도 동일한 입력 문제 발생
- 오른쪽 하단 화살표가 계속 켜져 있음
- 영어도 입력이 안 됨
- 메모장(텍스트 에디터)에 작성 후 복사 붙이기로 작업 중

프로젝트 정보:
- 프로젝트명: Aramed_AI
- 위치: ~/Documents/Aramed_AI
- 목적: LaTeX 수학 문제/해설을 DeepSeek 형식으로 변환하고 Notion과 연동
- 완료된 작업: 기하_2024학년도_현우진_드릴_P5까지 완료

요청:
1. 프로젝트 상태를 확인해주세요
2. 환경 설정(.env, 패키지 등)을 확인해주세요
3. 다음 작업을 시작할 수 있는지 확인해주세요

참고:
- Windows에서 작업하던 프로젝트를 Linux로 이어가려고 합니다
- 입력 문제로 인해 메모장에 작성해서 복사 붙이기로 작업하고 있습니다
- 이 방식으로도 효율적으로 작업할 수 있도록 도와주세요

어떻게 시작하면 좋을까요?
```

### 2단계: 프로젝트 상태 확인

**Cursor 답변을 확인한 후, 다음 질문:**

```
프로젝트 상태 확인 결과를 바탕으로:

1. 필요한 환경 설정이 있으면 알려주세요
   - .env 파일 설정
   - 패키지 설치
   - 환경 변수 설정

2. 다음 작업을 시작할 준비가 되었는지 확인해주세요

3. 작업을 시작할 수 있으면 다음 단계를 알려주세요
```

### 3. 단계별 진행
- 복잡한 작업은 단계별로 나눠서 요청
- 각 단계마다 확인 후 다음 단계 진행

## 텍스트 에디터 추천

**Linux에서 사용 가능한 텍스트 에디터:**
- `gedit`: GUI, 사용하기 쉬움
- `nano`: 터미널, 간단함
- `vim`: 터미널, 강력함
- `kate`: KDE, 기능 많음
- `mousepad`: Xfce, 가벼움

**추천:**
- GUI가 필요하면: `gedit`
- 터미널에서: `nano` (간단) 또는 `vim` (고급)
