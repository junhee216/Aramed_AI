# 수1 P2 문제 누락 원인 분석

## 디버깅 결과

### ✅ 매칭되는 패턴
- 문제 1번: 패턴 매칭 성공 (True)
- 문제 8번: 패턴 매칭 성공 (True)  
- 문제 9번: 패턴 매칭 성공 (True)

### ❌ 매칭 실패하는 패턴
- 문제 2번: 패턴 매칭 실패 (False)
- 문제 6번: 패턴 매칭 실패 (False)
- 문제 11번: 패턴 매칭 실패 (False)

## 구체적인 원인

### 1. 문제 1번 누락 원인
- **패턴**: `r'(\\\$a>1\\\$ 인 상수.*?\[4점\])(.*?)(?=\\\\section)'`
- **문제점**: 
  - `(?=\\\\section)`에서 `\\\\section`을 찾는데, 실제로는 `\\section`일 수 있음
  - 또는 조건문 `if len(question) > 50`에서 걸러졌을 가능성

### 2. 문제 2번 누락 원인
- **패턴**: `r'(함수 \$f\(x\)=\\log.*?구하시오\. \[4점\])'`
- **문제점**: 
  - `\$f\(x\)`에서 `\(`는 이스케이프된 괄호를 찾는 것이 아니라 실제 `(`를 찾아야 함
  - 실제 body에서는 `\$f(x)=\\log` 형태
  - **수정 필요**: `\$f\(x\)` → `\$f\(x\)` 또는 `\$f\(x\)` → `\$f\(x\)`

### 3. 문제 6번 누락 원인
- **패턴**: `r'10\\\\\s*(\\\$a>1\\\$ 인 실수.*?\[4점\])'`
- **문제점**: 
  - `10\\\\`는 찾았지만, 그 다음 `\\\$a>1\\\$` 패턴이 매칭되지 않음
  - 실제 body에서는 `10\\\\` 다음에 `\$a>1\$` 형태
  - **수정 필요**: `\\\$` → `\$` (body는 이미 `\begin{document}` 이후이므로)

### 4. 문제 8번, 9번 누락 원인
- 패턴은 매칭되지만, 다른 문제와 겹치거나 조건문에서 걸러졌을 가능성
- 문제 7번이 여러 문제를 합쳐서 추출한 것으로 보임

### 5. 문제 11번 누락 원인
- **패턴**: `r'(곡선 \$y=\\log.*?\[4점］)'`
- **문제점**: 
  - `\[4점］`에서 `］`는 전각 문자 (U+FF5D)
  - 패턴에서는 반각 `]`를 찾고 있음
  - **수정 필요**: `\[4점］` → `\[4점[\]］]` 또는 `\[4점[\]］]`

## 해결 방법

1. **문제 1번**: `(?=\\\\section)` → `(?=\\section)` 수정
2. **문제 2번**: `\$f\(x\)` → `\$f\(x\)` 수정 (괄호 이스케이프 제거)
3. **문제 6번**: `\\\$` → `\$` 수정
4. **문제 11번**: `\[4점］` → `\[4점[\]］]` 수정 (전각/반각 모두 매칭)
